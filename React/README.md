## Reactコンポーネントクラス
propsパラメータを受け取りrenderメソッドを介して表示するビューの階層を返す。stateには

### props
（「プロパティ」の略）

### render
画面に表示したいものの説明を返す
Reactはこれを受け取り結果を表示する

### JSX
構文。
例：React.createElement('div') でdev要素を生成できる。

### setState
値を設定する。また、中のコンポーネントも自動的に更新する。

### super(props);
一旦気にしなくて良い。新しいバージョンでは書かなくて良い?
